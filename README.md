# B-ze
B-ze is a basic DNA or RNA analyzer tool, you can ask for the length of a sequence, an specific nucleotide or to transcript DNA into RNA.


How to use:


-B-ze.py:


  In terminal type: ./B-ze.py [your file]

 
  The program will give you all the parameters.

  
-B.ze.1.1.py (update):


  In terminal type: ./B-ze.1.1.py [your file]

  
  The programs ask the process you wanna run, type: 1, 2, 3, 4 or 5

  
  The program ask the file type, type: yes (1) no (2)

  
  Then you'll have the result of the process you choosed
